# Chapter 2: 안 쓰는 코드
안 쓰는 코드는 뭔가 발견하기가 어렵지는 않은 것 같긴 하다. 보통 메소드 하나가 주석처리 되어 있기도 하고, 작성 되어 있는 함수나 메소드를 다른 곳에서 참조 혹은 호출하는지 확인 하는 기능을 IDE나 코드 에디터에서 제공을 하기 때문이다. 그 외에 잠시 디버깅 용도로 넣은 로그 찍는 코드도 찾기는 쉬워서 이런건 아마 쉽게 찾아다 지울 수 있지 않을까 싶다. 삭제 해도 괜찮은가에 대해서는 통으로 주석 처리된 것이야 쉽게 판단이 되겠지만, 다른 곳에서 호출 안하는 함수는 좀 판단이 어려울 수도 있겠다. 내가 작성한 거면 괜찮겠지만, 다른 동료가 작성 한 것이면 나중에(?) 다른곳에서 호출 해서 사용하려고 작성 했다가 까먹은 것일지도(?)
