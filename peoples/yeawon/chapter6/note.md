## 6장. 응집도를 높이는 배치

- 코드의 순서를 바꿔서 **변경할 요소들을 가까이 둬라**
- 한 파일에서 두 루틴에 결합도가 있으면 **서로 옆에 둬라**
- 두 파일에 결합도가 있으면 **같은 디렉토리**에 넣어라
- 결합도가 있는 코드를 **같은 코드 저장소**에 넣은 후 변경하라
- if, `결합도 제거(decoupling) 비용 + 변경 비용 < 결합도(coupling)에 따른 비용 + 변경 비용` → 결합도를 제거하라. 그러나, 여러 이유로 결합도 제거가 어려울 수 있다
- **응집도를 높이는 순서로 정리**하면, 더 쉽게 변경할 수 있다
- 응집도가 좋아지면 결합도 역시 덩달아 좋아짐
